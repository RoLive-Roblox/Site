Bot Commands

  • Track a game/games*
    r!trackgame [Place ID]...
    e.g. → r!trackgame 1224212277 5036207802 1064681037
  • Track a group/groups*
    r!trackgroup [Groupd ID]...
    e.g. → r!trackgroup 7976031 8994740
  • Display your current trackers
    r!mytrackers
  • Remove a tracker
    r!untrack [Tracker ID]
    Get the 'Tracker ID' by running the 'r!mytrackers' command
  • Display all bot commands
    r!help

                        * You will be able to select the tracker type from the bot using reactions.
